# Hola mundo

Hola mundo desde el proyecto de agentes.

Este archivo existe para probar la herramienta `file_reader`.
La idea es que el agente pueda leer un archivo de texto simple,
devolver su contenido al LLM y usar esa observacion para responder.

Contenido de prueba:

- linea uno: el agente recibe una ruta relativa.
- linea dos: la tool valida que el archivo este dentro de `sample_files`.
- linea tres: la tool devuelve el contenido como texto UTF-8.

Fin del archivo.
